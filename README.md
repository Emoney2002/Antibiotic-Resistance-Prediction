# Antibiotic-Resistance-Prediction
eveloped a machine learning pipeline to predict multi-drug resistance (MDR) in bacterial infections using patient clinical and microbiological data. Multi-drug resistance was defined as resistance to three or more antibiotics, consistent with standard antimicrobial resistance definitions. The project included feature engineering, missing data handling, and model development using Logistic Regression, Random Forest, and XGBoost.

Model performance was evaluated using ROC-AUC and classification metrics, and feature importance analysis was conducted to identify key predictors of antibiotic resistance.
